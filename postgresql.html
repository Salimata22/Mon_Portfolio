Gestion des données avec PostgreSQL

Ce projet met en œuvre un système de gestion de base de données relationnelle avec PostgreSQL. L'objectif est de concevoir une base de données performante pour une application web, incluant la création et l'optimisation des tables, la gestion des relations entre les entités et l'exécution de requêtes complexes pour extraire et analyser des données.

Technologies utilisées :
SGBD : PostgreSQL
Interface utilisateur : HTML/CSS, JavaScript
Back-end : Flask

Ce projet a été réalisé dans le cadre d'une mission académique.

Accédez au code source sur GitHub.
