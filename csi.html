Site de gestion d'un club nautique

Conception et développement d'un système complet de gestion pour un club nautique. Le projet comprend la planification des cours, la gestion des locations d'équipements, et le suivi des stocks. Le système intègre une interface utilisateur ergonomique et une base de données PostgreSQL pour assurer la performance et la sécurité des données.

Technologies utilisées :
Backend : PHP, PostgreSQL
Frontend : HTML, CSS, JavaScript

Contexte académique :
Réalisé dans le cadre du Master 1 MIAGE (2024-2025) pour le cours CSI Méthodes et CSI Optimisation.

Voir le code source sur GitHub

Points forts du projet :

  • Technologies utilisées : Backend en PHP, base de données PostgreSQL, et front-end en HTML, CSS, JavaScript.
  • Gestion des utilisateurs : Système de rôles et droits (propriétaire, gestionnaire, moniteurs), avec une sécurité renforcée et une gestion centralisée.
  • Fonctionnalités clés : Planification des cours de voile, gestion des forfaits, suivi des pannes du matériel, statistiques sur les activités du club, et gestion des campings partenaires.
  • Optimisation : Utilisation de triggers, procédures stockées, et index pour maintenir l’intégrité des données et améliorer les performances des requêtes.
  • Modélisation : UML pour la conception du système, diagrammes d'activités et de classes pour structurer les fonctionnalités.
